## Definition
Let $I$ be an [ideal]({{< ref ideals.md >}}) of the [polynomial ring]({{< ref poly_rings.md >}}) $\pring{k}{x_1, \dots, x_n}$. The **radical of $I$** is the set $$\sqrt{I} = \{f \mid f^m \in I \text{ for some integer } m \geq 1\}.$$

## Properties
If $I \subset \pring{k}{x_1, \dots, x_n}$ is an ideal, then $\sqrt{I}$ is also an ideal in $\pring{k}{x_1, \dots, x_n}$ and contains $I$. 

If $I \subset \pring{k}{x_1, \dots, x_n}$ is an ideal, then $\sqrt{I}$ is a [radical ideal]({{< ref radical_ideals.md >}}).
